# Protocole de discovery distribué

## Objet

Les collectors récupèrent des jobs, exécutent des scans bornés, publient des observations signées et ne conservent pas de secrets après usage.

## Format attendu

- Objectif.
- Périmètre.
- Contrats d’entrée/sortie.
- Règles de sécurité.
- Règles de performance.
- Conditions d’erreur.
- Tests obligatoires.
- Critères d’acceptation.

## Application

Ce RFC est normatif pour toute implémentation liée au sujet. Toute divergence doit être approuvée par ADR.
