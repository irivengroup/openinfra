# Matrice de traçabilité — lecture

## Principe

La matrice de traçabilité relie les exigences, les cas d’usage, les tests et les preuves attendues. Elle est fournie au format CSV dans `11-Matrices/Traceabilite.csv`.

## Règle de réception

Aucune exigence N1 ne doit rester sans test, preuve ou cas d’usage associé. Les exigences N2 peuvent être rattachées à des tests différés seulement si la roadmap précise la phase de réalisation.
