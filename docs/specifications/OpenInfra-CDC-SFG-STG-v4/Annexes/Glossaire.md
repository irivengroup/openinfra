# Glossaire

| Terme | Définition |
| --- | --- |
| ABAC | Attribute-Based Access Control : contrôle d’accès basé sur des attributs comme tenant, site, environnement ou criticité. |
| ADR | Architecture Decision Record : décision d’architecture versionnée et argumentée. |
| BFF | Backend For Frontend : couche API adaptée aux besoins de l’interface utilisateur. |
| BRIN | Block Range Index : index PostgreSQL adapté aux grandes tables naturellement corrélées. |
| CCTP | Cahier des Clauses Techniques Particulières. |
| CdCF | Cahier des Charges Fonctionnel. |
| CIDR | Classless Inter-Domain Routing : notation des préfixes IP. |
| DCIM | Data Center Infrastructure Management. |
| DDI | DNS, DHCP and IPAM. |
| EVPN | Ethernet VPN. |
| HA | High Availability. |
| IPAM | IP Address Management. |
| ITAM | IT Asset Management. |
| MFA | Multi-Factor Authentication. |
| PCA | Plan de Continuité d’Activité. |
| PITR | Point-In-Time Recovery. |
| PRA | Plan de Reprise d’Activité. |
| RBAC | Role-Based Access Control. |
| RPKI | Resource Public Key Infrastructure. |
| SFG | Spécifications Fonctionnelles Générales. |
| STG | Spécifications Techniques Générales. |
| SPOF | Single Point of Failure. |
| VRF | Virtual Routing and Forwarding. |
| VXLAN | Virtual Extensible LAN. |
